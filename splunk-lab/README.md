# Splunk Lab

A dedicated SIEM environment used to develop skills in:

- Log collection and ingestion
- Endpoint telemetry analysis
- Detection engineering
- Alert development
- Dashboard creation
- Threat hunting
- Attack simulation

## Projects

### Splunk SIEM Deployment and Endpoint Telemetry Analysis

Built a Splunk Enterprise SIEM environment using Ubuntu Server, Windows 10, Sysmon, and the Splunk Universal Forwarder.

**Status:** Complete

[View Project](./splunk-siem-deployment-and-endpoint-telemetry-analysis)

### Detection Engineering & Alert Validation

Developed and validated custom Splunk detections and alerts using Sysmon and Windows Security logs to identify suspicious activity, including encoded PowerShell execution, PowerShell web requests, LOLBin abuse, account enumeration, and failed logon attempts.

**Status:** Complete

[View Project](./detection-engineering-and-alert-validation)

### SOC Dashboard Development

Built a SOC dashboard in Splunk to visualize endpoint telemetry, process execution, authentication activity, log source distribution, and detection performance. Tuned dashboard visualizations to reduce noise and improve analyst visibility into security-relevant events.

**Status:** Complete
